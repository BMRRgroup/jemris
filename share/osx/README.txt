JEMRIS version 2.8.2

IMPORTANT NOTES !

1. Add "/usr/local/share/jemris/matlab" to the MATLAB search path to run the JEMRIS-GUIs (JEMRIS_seq, JEMRIS_sim, JEMRIS_txrx)

2. Add "/usr/local/bin" to the PATH environment variable to run jemris from the command line 

3. The parallel version (pjemris) requires to install mpich2 via Homebrew (http://brew.sh/).


visit http://www.jemris.org for more information
